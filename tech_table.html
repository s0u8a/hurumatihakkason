使用技術・API ロゴ アプリでの役割
Cloudinary
クラウド画像管理 API
 ユーザーが撮影した写真を一時的にクラウドへ保存し、AIに渡すための画像URLを生成するために使用しています。
Google Cloud
Vision API
AI 画像解析 API
写真をAIが解析し「ランドマーク」や「写っている物」を特定。写真判定だけで自動的にスタンプを付与する仕組みの心臓部です。
Leaflet
地図描画ライブラリ
 マップ画面で、観光スポットのピン📍を立てたり、スワイプで動かせるインタラクティブな地図画面を描画しています。
OpenStreetMap
オープンソース地図データ
 Leafletが地図を描画する際に使っている「ベースの道路・建物の地図画像データ」です。無料で利用しています。

Vanilla JS
フロントエンド言語
画面構造、デザイン、動作ロジックの中核。今回は動作が重くなるフレームワークを使わず標準技術のみで構築し、スマホでの動作を軽くしています。
